Project Description

The project was created during a Web Application Development (COMP 229) course at Centennial College using as main technologies Node.JS and Angular.

Other technologies have been used such as:

  • MongoDB Atlas for the database
  • Firebase and Heroku for hosting
  • Github for file management

    • The project was composed of five development phases.

    The first phase was project planing. The goal was to decide what type of platform we should build. Following the idea, we started working on the wireframe and external documentation regarding the execution of the project.

    The second step in the project was to initialize every components required to the execution. For instance, we initialize the server and client application, the database account creation, and the github repository. After initializing each components, we connected each one of them together. Some basic pages were designed and the initiale routing was created.

    The third phase included everything linked to the survey management tools like survey creation, survey editing, survey deletion and duplication. This stepped allowed user to manage any survey on the platform and save it to MongoDB Atlas.

    The fourth step included everything linked to the user management tools like user login, user registration and user logout. Adaptation was made to the survey components to be managed by the user who created it and anyone wanting to create a survey are now required to be registered to be able to create or edit surveys.

    The fifth is the last step allowing us to fix the page visuals and improve the website. More testing and revision was made. Also published the full website on the web.

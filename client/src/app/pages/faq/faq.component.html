Project Description

The project was created during a Web Application Development (COMP 229) course at Centennial College using Node.JS and Angular. This project is based on a functionable Survey site. It will allow the user to register and create an account where they can manage all the Surveys including Add, Edit and Deleting the surveys.



FAQ

How can I see the result for my survey?

Once the survey is over, you can access the results under your account.

Who can view the survey results?

Survey results are kept visible to only the survey's owner or they can be shared by the survey owner.

Who can add a survey to the site or page layout?

Anyone with edit access or registered account to the site can add a survey card to the layout.

What survey template is available?

When creating a new survey, there is Questionnaire template with multiple options available to help you get started.

Where can you create a survey in Codek Surveys?

You can create a new survey by clicking on "New Survey" button on your personalized survey list page when logged in.
